

# FoodEase – Online Food Delivery Management System

**FoodEase** is a full-stack web application developed as a **Database Systems** project that provides a complete online food ordering and delivery platform. The system connects customers, restaurants, delivery staff, and administrators through a role-based management system, allowing users to browse restaurants, place food orders, manage deliveries, and monitor platform activities efficiently.

The application features secure user authentication, restaurant and menu management, shopping cart functionality, online order processing, payment management, delivery tracking, customer feedback, and an administrative dashboard. The backend is powered by a relational MySQL database designed using normalization principles, ensuring data integrity, scalability, and efficient query execution.

## Features

* Secure user registration and login
* Role-based access control (Admin, Customer, Restaurant Owner, Delivery Staff)
* Restaurant profile management
* Food menu management (Add, Update, Delete)
* Browse and filter restaurants and food items
* Shopping cart and order placement
* Order status tracking
* Payment management
* Delivery staff assignment
* Customer reviews and ratings
* Admin dashboard for managing users, restaurants, and orders
* Reports and analytics

## User Roles

### Customer

* Register and log in
* Browse restaurants and menus
* Add food items to cart
* Place and track orders
* Provide ratings and feedback

### Restaurant Owner

* Manage restaurant profile
* Add, edit, and delete food items
* Update menu availability
* View customer feedback

### Delivery Staff

* View assigned deliveries
* Update delivery status
* Manage completed orders

### Administrator

* Manage all users
* Manage restaurants and menus
* Monitor orders and deliveries
* Generate reports and analytics

## Project Workflow

1. Users register and log in based on their role.
2. Customers browse restaurants and food menus.
3. Food items are added to the shopping cart.
4. Customers place orders and select a payment method.
5. Orders are assigned to delivery staff.
6. Customers track order progress in real time.
7. After delivery, customers submit ratings and reviews.
8. Administrators manage the overall platform and generate reports.
